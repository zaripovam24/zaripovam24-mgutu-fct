# Проект от ФЦТ МГУТУ
Разрабатывается студентами Факультета цифровых технологий МГУТУ им. К.Г. Разумовского

## Научная основа
Проект продолжает исследования, проведённые в МГУТУ:
- ВКР «Разработка мобильного приложения "StudyKids" на базе ОС Android» направление подготовки 09.03.01 (2025)  
- ВКР «Разработка компьютерной игры для внедрения геймификации» (2025)
Ключевой метод: адаптация методов геймификации и анализа целевой аудитории для платформы с иностранным контентом.

## Как помочь?
Мы рады контрибьюторам любого уровня! Начать легко:

1. Сделайте форк репозитория
2. Клонируйте свой форк:  
   `git clone https://github.com/zaripovam24/zaripovam24-mgutu-fct.git`
3. Установите зависимости:  
   `pip install -r requirements.txt`
4. Запустите тесты:  
   `python -m pytest tests/`
5. Внесите правки → создайте PR

Вопросы? Откройте Issue с меткой `question`.
